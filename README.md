# Edu_Sync

Edu_Sync V0 is a frontend-first prototype for a lightweight, QR-based attendance system designed for colleges that already use an existing ERP or internal database.

This version focuses on clarity, structure, and long-term maintainability, not on implementation logic. All functionality is intentionally deferred to later phases.

Project Philosophy

Edu_Sync V0 follows three core principles:

1. Speed over features
   Attendance should take seconds, not minutes.

2. Simplicity over complexity
   Faculty and students should understand the system without training.

3. Compatibility over control
   Edu_Sync does not replace existing ERPs; it complements them.

This repository represents the UI surface only, built to support future backend and frontend expansion without refactoring.

What This Version Includes

Static landing page

Authentication UI (login, signup, forgot password)

Student dashboard UI

Teacher dashboard UI

Clean, modular HTML and CSS architecture

Explicit placeholders for future backend and JavaScript integration

What This Version Does NOT Include

JavaScript logic

Form validation

Authentication or authorization

Role-based routing

Backend integration

API calls

State management

Analytics or charts

All behavior is intentionally excluded.
